# Задача 5

Напишете функция `sum_digits`, която пресмята сумата на всички цифри между две числа.
Напр.: за числа 19 и 22 (т. е. 19, 20, 21, 22): (1 + 9) + (2 + 0) + (2 + 1) + (2 + 2) = 19

```python
>>> sum_digits(7, 8)
15
>>> sum_digits(17, 20)
29
>>> sum_digits(10, 12)
6
```
